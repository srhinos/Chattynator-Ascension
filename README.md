# Chattynator (3.3.5a)

A back-port of [Chattynator](https://github.com/TheMouseNest/Chattynator) — a full chat-interface replacement — to WoW 3.3.5a, for Project Ascension.

## Install

Drop the `Chattynator` folder into `Interface\AddOns\`, then type `/chatty` in-game to open settings.

## Slash commands

- `/chatty` — open settings
- `/chatty diag` — compatibility/status report
- `/chatty dump` — chat-frame introspection
